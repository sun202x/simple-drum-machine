## Web Audio API
Web Audio API는 일반적인 AudioElement로 구현할 수 없는 복잡하고 다양한 오디오 기능을 구현하기 위해 사용된다.
- 게임
- 화상회의
- 음악교육
- 음악제작
- 입체음향
